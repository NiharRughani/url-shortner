# url-shortner
 web application that converts long URLs into short, shareable links with automatic redirection.


## Features
- Shorten long URLs
- Automatic redirection
- Unique short code generation
- MongoDB-based persistent storage
- Simple and clean UI

- ## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- ODM: Mongoose

## How It Works
1. User submits a long URL
2. Backend generates a unique short code
3. URL is stored in the database
4. Short URL redirects to the original link
